WPF Z80 Scenario Editor

This tool enables creation of scenarios for the "Zelda" game engine. It is able to simulate the game using Wabbitemu as the backend by intercepting drawing calls and peering into the game engine data. This allows the game to run not only on a Z80 calculator, but in color on a Windows PC as well.  This allows the project to reach a much wider audience, while retaining the novelty of still running on a 6 MHz Z80.
