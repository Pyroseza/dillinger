Description:

The .fill command is very similar to the .block command. It can be used identically, but has the extra option of allowing you to specify a value to fill. For more information on how to properly fill values look at the .block command.

Usage:

.fill size, [value]

Parameters:

Size

The size of the data you wish to insert.

Value

The value to insert as data. It is only possible to specify one 8-bit value here
