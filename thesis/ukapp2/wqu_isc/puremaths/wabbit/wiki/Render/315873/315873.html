interface IKeypad : IDispatch

This interface implements all of the keypad peripheral features.

Methods

  • VOID PressKey(CalcKey Key)

Presses a calculator key

  • VOID ReleaseKey(CalcKey Key)

Releases a calculator key

  • Boolean IsKeyPressed(CalcKey Key)

Reports whether or not a key is currently pressed

Enumerations

  • CalcKey
KEY_UP = 0
KEY_LEFT = 1
KEY_RIGHT = 2
KEY_DOWN = 3
